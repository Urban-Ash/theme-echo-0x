# Echo 0x 1.1.0

Echo 0x `1.1.0` 是一次围绕辅助页系统和整体一致性的整理版本，重点是把标签、分类、归档、友情链接这些页面统一进同一套终端风骨架，并同步清理项目文档。

## 亮点

- 统一 `tags / categories / archives / links` 的辅助页视觉语言
- 为标签页补齐面板化结构、热度说明和更统一的节奏
- 分类页改为树状索引，归档页改为时间流式列表
- 友情链接页支持更清晰的终端式分组展示
- 修正文档入口、安装说明、版本说明和开发指南

## 本次调整

### 页面层

- `tags.html`：收口到 `aux-page` 体系，保留标签权重热度
- `categories.html`：改为树状终端索引
- `archives.html`：改为年份分组的时间线流
- `links.html`：支持插件模式与单页模式双兼容

### 框架层

- 稳定 PJAX 容器宽度与撑满行为
- 保持页脚透明背景与底部落位一致
- 继续沿用全局共享头部、页脚和页面壳层

### 文档层

- 根 `README` 改为项目总览入口
- `docs/README` 改为用户使用指南
- `docs/development` 更新为当前维护流程
- `docs/CHANGELOG` 追加 `1.1.0` 版本记录

## 安装

1. 下载 `1.1.0.zip`
2. 进入 Halo 后台“主题”页面
3. 上传压缩包并启用 `Echo 0x`

## 备注

- 本版本延续 Halo `2.25.0+` 要求
- 升级后建议重新上传 ZIP，以确保静态资源版本生效
