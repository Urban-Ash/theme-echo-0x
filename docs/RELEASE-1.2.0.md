# Echo 0x 1.2.0

Echo 0x `1.2.0` 是一次围绕文章页、辅助页一致性和样式维护性的正式整理版本。这个版本一方面收口了文章页标题区、代码块和阅读流，另一方面继续清理并合并了辅助页与详情页里重复的样式定义，让后续维护更轻。

## 亮点

- 文章页标题区、封面和代码块样式继续向终端风参考稿靠拢
- 保留当前认可的文章页头部，同时把正文阅读流拉回更稳的结构
- `tag / category` 详情页的重复 CSS 合并为统一样式分支
- 辅助页空状态抽成统一的 `empty-state` 基类
- 项目文档、版本入口和发布说明统一升级到 `1.2.0`

## 本次调整

### 文章页

- `post.html`：保留当前认可的文章头部，继续优化正文区与导航结构
- `style.css`：重做文章页标题、摘要、标签和代码块样式
- 清理旧的文章页残留选择器，减少 `.article-cover` 一类冲突源

### 样式层

- 合并辅助页公共 shell / frame 样式
- 合并 `tagentry-*` 与 `catentry-*` 的重复详情页样式
- 将多页空状态收口到统一的 `.empty-state` 基类
- 删除一批高置信度的旧样式残留，减轻 `style.css` 冗余

### 文档层

- 根 `README` 更新到 `1.2.0`
- `docs/README`、`docs/CHANGELOG`、`docs/development` 同步当前版本状态
- 新增 `docs/RELEASE-1.2.0.md` 作为正式发布说明

## 安装

1. 下载 `1.2.0.zip`
2. 进入 Halo 后台“主题”页面
3. 上传压缩包并启用 `Echo 0x`

## 备注

- 本版本继续要求 Halo `2.25.0+`
- 升级后建议重新上传 ZIP，以确保静态资源版本号和浏览器缓存同步刷新
