# Echo 0x 使用指南

本文档面向主题使用者，介绍安装方式、主要页面、后台配置和常见使用方式。

## 主题简介

Echo 0x 是一个为 Halo 2.25+ 设计的暗色终端风主题，适用于安全、AI、开发和技术写作场景。它强调统一的终端气质、数学公式背景和结构化排版，适合长文、索引页与独立页面共同使用。

## 适用版本

- Halo：`2.25.0+`
- 主题当前版本：`1.3.0`

## 安装方式

### 通过 Halo 后台上传

1. 下载 Release 中的主题压缩包，例如 `1.3.0.zip`
2. 进入 Halo 控制台的“主题”页面
3. 点击“安装主题”并上传压缩包
4. 安装完成后启用 `Echo 0x`

### 通过目录部署

1. 获取仓库源码
2. 将 `theme-echo-0x` 目录放入 Halo 的 `templates/themes` 目录
3. 返回控制台启用主题

## 页面与模板

### 默认页面

- 首页：Halo 站点名称、可配置标语、数据概览、最新文章列表
- 文章页：公式占位封面、目录、元信息、正文与文章导航
- 标签页：终端风标签云与热度分布
- 分类页：树状分类索引
- 归档页：时间流式归档
- 作者页：作者资料与文章列表
- 错误页：`404` 和通用错误页

### 自定义页面模板

#### About

在 Halo 后台创建独立页面时选择 `About` 模板，可用于个人介绍、项目说明或履历展示。

#### Links

`Links` 模板支持两种模式：

1. 普通独立页：直接渲染编辑器正文内容
2. 插件友链页：若启用了友情链接插件并传入分组数据，会自动渲染终端风链接列表

插件友链页会在浏览器端对每个 HTTP(S) 地址进行两次轻量探测，并显示以下状态：

- `online`：两次探测均成功且响应较快
- `warn`：响应较慢、结果不稳定，或因 HTTPS 页面访问 HTTP 地址而无法可靠验证
- `idle`：两次探测均失败、超时或协议不受支持

检测结果受浏览器网络策略、跨域规则、代理和目标站点限流影响，只代表当前访客环境下的可达性参考，不是服务端监控结果。

## 后台配置说明

### 首页 `home`

| 配置项 | 说明 | 默认值 |
| --- | --- | --- |
| 首页标语第一段 | 站点名称下方标语的第一段 | `break things.` |
| 首页标语第二段 | 站点名称下方标语的第二段 | `build systems` |
| 首页标语第三段 | 站点名称下方标语的强调段 | `that learn.` |
| 首页简介 | 首页介绍文本 | 网安工程师 x AI 全栈开发... |
| 首页索引数据 | 最多 4 项，可排序，可清空 | `posts, visits, comments, categories` |
| 首页文章卡片强制使用主题封面 | 开启后首页文章卡片忽略文章自带封面，统一显示主题公式封面；关闭后有文章封面时显示文章封面，没有时回退到主题公式封面 | `false` |

支持的数据项包括：`posts`、`visits`、`comments`、`categories`、`upvotes`、`halo`、`lang`、`title`、`url`、`registration`。

首页最醒目的站点名称读取 Halo 的站点标题，可在 Halo 的站点设置中修改，不由上述三段标语控制。

### 样式 `style`

| 配置项 | 说明 | 默认值 |
| --- | --- | --- |
| 辅助色调 | 主题强调色 | `#e63946` |
| 明暗模式 | `system / light / dark` | `system` |
| 页面切换特效 | 是否启用轻量 PJAX 淡入切换 | `false` |
| 封面风格 | `random / ai / crypto / original` | `random` |
#### 封面风格说明

- `ai`：Sigmoid、神经网络、损失函数、注意力矩阵等图形
- `crypto`：XOR、哈希、Feistel 网络、密钥流等图形
- `original`：Lissajous 曲线、扫描线、几何分形等图形
- `random`：按标题哈希或默认索引自动选择风格

主题会将辅助色同步到公式封面和插件组件。选择“跟随系统”时，页面会实时响应系统明暗模式变化。

### 导航栏 `nav`

| 配置项 | 说明 | 默认值 |
| --- | --- | --- |
| 显示时间 | 是否显示顶部时钟 | `true` |
| 显示访客 IP | 是否显示访客 IP | `false` |
| 访客 IP 查询接口 | 返回纯文本或 JSON 的接口 | `https://api.ipify.org?format=json` |
| 显示页面路径 | 是否显示当前路径 | `true` |
| 导航菜单 | 选择 Halo 菜单；为空时回退主菜单或默认导航 | 空 |

### 插件 `plugin`

| 配置项 | 说明 | 默认值 |
| --- | --- | --- |
| 启用搜索组件入口 | 控制导航栏里的 `search` 入口是否显示；仅在已安装搜索插件时生效 | `true` |
| 启用评论组件 | 控制文章页、独立页、About、Links 单页模式下的评论区是否显示；仅在已安装评论插件时生效 | `true` |

### 文章页 `post`

| 配置项 | 说明 | 默认值 |
| --- | --- | --- |
| 显示章节导航 | 是否显示文章页目录 | `true` |
| 强制使用主题封面 | 开启后忽略文章自带封面，统一显示主题公式封面；关闭后有文章封面时显示文章封面，没有时回退到主题公式封面 | `false` |

文章没有主图时，主题会根据标题稳定选择公式封面。文章配置了主图时，公式 SVG 会先作为加载占位显示，图片加载完成后再平滑替换；图片加载失败时继续保留公式封面。

### 页脚 `footer`

| 配置项 | 说明 | 默认值 |
| --- | --- | --- |
| 显示左侧 cat | 是否显示左侧提示文案 | `true` |
| 左侧 cat 文案 | 页脚左侧提示内容 | `CC BY-NC-SA 4.0` |
| 左侧 cat 跳转链接 | 点击提示后的链接 | CC 协议地址 |
| 显示 pause fx 按钮 | 是否显示背景动画暂停按钮 | `true` |
| 显示 RSS | 是否显示 RSS 链接 | `true` |
| RSS 显示文字 | 页脚 RSS 文案 | `rss` |
| RSS 链接 | RSS 地址 | `/rss.xml` |
| 显示 GitHub | 是否显示 GitHub 链接 | `true` |
| GitHub 显示文字 | GitHub 文案 | `github` |
| GitHub 链接 | GitHub 地址 | `https://github.com/` |
| 显示 Email | 是否显示 Email 链接 | `true` |
| Email 显示文字 | Email 文案 | `email` |
| Email 链接 | Email 地址 | `mailto:hello@example.com` |

页脚项目均可单独显示或隐藏；链接文字和目标地址也可以分别修改。

## Halo 插件兼容

- 搜索：安装 Halo 搜索组件后，导航栏可显示 `search` 入口
- 评论：文章、普通独立页、About 与单页模式 Links 可渲染 Halo 评论组件
- 图库：正文区域带有 Halo LightGallery 兼容类，已安装对应组件时可接管图片浏览
- 页脚：保留 `<halo:footer />` 扩展点，供插件注入前端资源

## 使用建议

- 如果你希望统一页面风格，优先补齐标签、分类、归档、友情链接这些辅助页内容
- 如果你使用友情链接插件，建议为不同链接建立分组，以便 `Links` 页面展示筛选视图
- 如果你希望主题更轻，先从关闭访客 IP 和页面切换特效开始
- 升级版本后建议重新上传 ZIP，避免静态资源缓存影响效果
