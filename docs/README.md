# Echo 0x 主题文档

## 主题介绍

Echo 0x 是一个为 Halo 2.25+ 设计的深色终端风格主题，适用于网络安全、AI、技术笔记类博客。

### 特性

- 深色/浅色模式切换
- 封面风格自定义（AI/密码学/原风格/随机混合）
- 实时计算的数学动画封面
- 页面切换特效（可配置）
- 首页数据面板（可配置显示项）
- 章节导航（文章页）
- 标签/分类/归档页面
- 友情链接页面（支持插件）
- 访客 IP 显示（可选）
- 时钟显示（可选）
- 响应式设计，适配各种设备
- 统一的资源版本管理

## 安装

### 1. 上传安装

下载主题压缩包（如 `1.0.0.zip`），进入 Halo 控制台：
1. 左侧菜单 → 主题
2. 点击「安装主题」
3. 选择「上传」选项，上传压缩包
4. 安装完成后点击「启用」

### 2. 直接放入目录

将解压后的 `theme-echo-0x` 目录直接放入 Halo 的 `templates/themes` 目录中，然后在控制台启用。

## 主题配置

### 首页配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 首页标题第一行 | Hero 区域主标题第一行 | break things. |
| 首页标题第二行 | Hero 区域主标题第二行（带描边） | build systems |
| 首页标题第三行 | Hero 区域主标题第三行（强调色） | that learn. |
| 首页副标题 | Hero 区域简介文本 | 网安工程师 x AI 全栈开发... |
| 首页数据面板 | 选择要显示的数据项（最多 4 项），可排序，清空则关闭 | posts, visits, comments, categories |

首页数据面板支持的项：
- posts：文章数
- visits：访问量
- comments：评论数
- categories：分类数
- upvotes：点赞数
- halo：Halo 版本
- lang：站点语言
- title：站点标题
- url：站点地址
- registration：注册开关

### 样式配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 辅助色调 | 主题强调色（红色系） | #e63946 |
| 明暗模式 | 跟随系统/明亮/黑暗 | system |
| 页面切换特效 | 是否启用切换时的扫描线动画 | true |
| 封面风格 | random（AI+密码学+原风格混合）/ ai（仅 AI 主题）/ crypto（仅密码学）/ original（仅原风格） | random |

#### 封面说明

- AI 风格：包含 Sigmoid 曲线、神经网络、损失函数、注意力矩阵等数学图形
- 密码学风格：XOR 运算、哈希函数、Feistel 网络、密文流转等
- 原风格：Lissajous 曲线、扫描线、同心圆、三角形分形等
- 随机：从所有风格中随机选择（按文章标题哈希或默认索引）

封面会根据文章标题自动匹配风格（如果配置了主题关键词）。

### 导航配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 显示时间 | 导航栏左侧是否显示实时时钟 | true |
| 显示访客 IP | 导航栏左侧是否显示访客 IP | false |

| 导航菜单 | 选择要使用的菜单，未选择则使用主菜单，主菜单空则使用默认 | （空） |

|--------|------|--------|

## 自定义页面模板

### About 页面

在 Halo 控制台创建独立页面时，选择「About」模板，该页面会使用文章页的样式但没有文章的元信息。

### Links 页面

有两种用法：

1. 作为普通独立页面：在编辑器中写内容，下方可配合友情链接插件
2. 配合插件链接：安装并配置 `plugin-links`，页面会自动显示分组的友情链接卡片

## 版本管理

主题版本号唯一从 `theme.yaml` 的 `spec.version` 读取，所有静态资源（CSS/JS）的版本会自动同步，避免浏览器缓存过期问题。
