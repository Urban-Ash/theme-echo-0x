# Echo 0x 1.0.0

`1.0.0` 是 Echo 0x 的首个正式发布版本。

这一版本面向 Halo 2.25+，围绕暗色终端风格、数学公式封面与技术内容表达进行了完整交付，可直接作为后续迭代的稳定基线使用。

## 发布亮点

- 提供统一的暗色终端视觉风格，适合安全、AI 与技术博客场景
- 支持实时计算的数学动画封面，覆盖 AI、密码学与原风格三类主题
- 完整适配 Halo 2.25+，包含首页、文章、标签、分类、归档、作者与独立页面模板
- 支持明暗模式切换、章节导航、首页数据面板与页面切换特效
- 统一通过主题版本字段管理资源版本，便于安装与缓存控制

## Release 内容

- `1.0.0.zip` 安装包
- `theme-echo-0x/` 主题源码
- `docs/` 用户与开发文档

## 安装说明

1. 下载本页附件中的 `1.0.0.zip`
2. 进入 Halo 后台，打开“主题”
3. 选择“安装主题”并上传压缩包
4. 安装成功后启用 `Echo 0x`

## 版本说明

- 本次发布将当前主题定义为新的 `1.0.0` 基线版本
- 仓库已完成初始整理，并补充基础忽略规则
- 后续功能迭代建议从 `1.0.1` 开始递增版本号
