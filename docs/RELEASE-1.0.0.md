# Echo 0x 1.0.0

Echo 0x 的首个正式发布版本，面向 Halo 2.25+，提供暗色终端风格、数学公式封面、可配置首页数据面板与 PJAX 页面切换体验。

## 亮点

- 暗色终端风格界面，适合安全、AI 与技术博客
- 实时计算的数学动画封面，支持 AI / 密码学 / 原风格混合
- Halo 2.25+ 适配，包含标签、分类、归档、作者页与独立页面模板
- 支持明暗模式切换、章节导航、访客 IP、顶部时钟与页面过渡特效
- 统一通过 `theme.yaml` 输出主题版本，便于资源缓存控制

## 本次包含

- `theme-echo-0x/` 主题目录
- `1.0.0.zip` 安装包
- `docs/` 使用文档与开发文档
- `AGENTS.md` 协作与视觉规范

## 安装方式

1. 下载 Release 附件中的 `1.0.0.zip`
2. 进入 Halo 后台，打开“主题”
3. 选择“安装主题”并上传压缩包
4. 安装后启用 `Echo 0x`
