# Echo 0x

> Dark terminal style Halo theme for security, AI and technical notes.

Echo 0x 是为 Halo 2.25+ 设计的暗色终端风格主题，支持明暗模式切换、实时公式动画封面、可配置数据面板和页面过渡特效，适合用于安全技术、AI 与开发类博客。

![preview](https://placehold.co/1200x600/0a0a0a/e63946?text=Echo+0x+Theme+Preview)

## ✨ 特性

- 🌙 明暗模式切换（跟随系统、明亮、暗色）
- 🎨 可配置强调色
- 🎯 支持三种封面风格 + 随机混合（AI 类、密码学类、原风格）
- 🔢 实时计算的数学动画封面
- 📊 首页可配置数据面板
- ⚡ 可开关页面过渡特效
- 📝 文章页支持章节导航
- 🔗 友情链接页面（支持插件）
- 📱 响应式布局
- 🏷️ 标签/分类/归档/作者页完整

## 📦 安装

### 方法一：通过 Halo 控制台上传
1. 下载 [最新 Release 附件 `1.0.0.zip`](https://github.com/Urban-Ash/theme-echo-0x/releases/latest)
2. 进入 Halo 后台 → 主题 → 安装主题
3. 选择「上传」，选择刚才下载的 zip 文件
4. 安装成功后点击「启用」

### 方法二：直接放入目录
1. 克隆或下载仓库
2. 把 `theme-echo-0x` 目录放入 Halo 的 `templates/themes` 目录
3. 在 Halo 后台点击「启用」
